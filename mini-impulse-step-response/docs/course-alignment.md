# Course Alignment: mini-impulse-step-response

## MIT (Massachusetts Institute of Technology)

**6.302 Feedback Systems** — Time-domain response, impulse/step characterization
- First/second-order system responses (Chapters 2-3)
- Time-domain specifications: tr, tp, ts, Mp (Chapter 3)
- Convolution and LTI system properties

**6.241 Dynamic Systems & Control** — State-space time response
- Matrix exponential for state transition
- Impulse response from state-space models

## Stanford University

**ENGR105 Feedback Control** — Transient response analysis
- First and second-order system step responses
- Design specifications in time domain
- Effect of poles and zeros on transient response

## UC Berkeley

**ME132 Dynamic Systems & Feedback** — Time-domain analysis
- Impulse and step response of mechanical systems
- Parameter identification from response data
- Damping ratio and natural frequency estimation

## Caltech

**CDS 110 Introduction to Control** — Fundamental time response
- Impulse response as fundamental system description
- Convolution integral for LTI systems
- Relationship between pole locations and time response

## ETH Zurich

**151-0591 Control I** — Zeitantwortanalyse (Time Response Analysis)
- Sprungantwort (step response) and Impulsantwort (impulse response)
- Time-domain performance criteria (Gutekriterien)
- PT1 and PT2 system responses

## Cambridge University

**3F2 Systems & Control** — Transient response
- First and second-order system analysis
- Time-domain specifications and design
- Discrete-time response and sampling

## Georgia Tech

**ECE 6550 Nonlinear Systems** — (Partial coverage)
- Impulse response for linearized systems
- Step response for equilibrium analysis

## Purdue University

**ECE 602 Lumped Systems** — Time-domain characterization
- Impulse and step response of lumped-parameter systems
- Convolution and superposition

## Tsinghua University (清华大学)

**自动控制原理 (Automatic Control Theory)** — 时域分析法
- 一阶系统时域分析 (First-order time-domain analysis)
- 二阶系统时域分析 (Second-order time-domain analysis)
- 动态性能指标 (Dynamic performance indices)
- 稳态误差分析 (Steady-state error analysis)
